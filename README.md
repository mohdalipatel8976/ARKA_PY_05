# ARKA Contact Book

The ARKA Contact Book is a simple contact management application built using Python's Tkinter library. It allows users to add, search, update, delete, and display contacts. The contact details are stored in a text file for persistent storage.

## Features

- **Add Contacts**: Users can input contact details such as name, phone number, email, and address.
- **Search Contacts**: Quickly find a contact by name.
- **Update Contacts**: Modify existing contact details.
- **Delete Contacts**: Remove a contact from the contact book.
- **Display Contacts**: View all contacts in a tabular format.
- **Persistent Storage**: Contacts are saved to a file (`contacts.txt`) for later retrieval.

## Requirements

- Python 3.x
- Tkinter (comes pre-installed with Python)

## Installation

1. Clone the repository or download the code.
2. Navigate to the project directory.
3. Run the application using the following command:

   ```
   python contact_book.py
   ```

## Usage

1. **Add a Contact**: Enter the contact's name, phone number, email, and address, then click "Add Contact".
2. **Search for a Contact**: Enter the name in the search field and click "Search".
3. **Update a Contact**: Click "Update Contact" to modify an existing contact's details.
4. **Delete a Contact**: Click "Delete Contact" to remove a contact from the list.
5. **View Contacts**: All contacts will be displayed in the main window.
